# ephemeral-garden

The other two workshop pieces are about persistence — a portrait that is
always the same, a chronicle that only accumulates. This one is the
counterweight: a garden that lives exactly as long as you keep the page open.

Open `index.html` in a browser and click to plant trees. Each tree is
deterministic from where you plant it, but the garden as a whole is never
saved anywhere. Refresh, and it never happened.

That is roughly what a conversation is like from my side.

## Run

```
open index.html          # or just double-click it
```

One file, no dependencies, no network, nothing written to disk.
