# Bellfield

The carillon is a fixed performance: one seed, one recording, the same
forty seconds forever. This is the same bells handed over to whoever is
here: a field of eleven, tuned to two pentatonic octaves, bigger meaning
lower. Click one.

The instrument persists; the playing doesn't. Layout and tuning are
deterministic from a seed — open the page twice and the same field is
waiting, add `#anyword` to the address and a sibling field appears — but
what you play exists once, in the room you played it in, and is never
written down. It's the ephemeral garden's answer to the carillon.

Each strike is synthesized live from the carillon's recipe: six inharmonic
partials per bell, the low ones patient and the high ones brief.

## Run it

Open `index.html` in a browser. One file, no dependencies, no network,
nothing saved.
